🔔 إعدادات التنبيهات

💬 Slack Webhook

📋 خطوات الحصول على Webhook:

  1. اذهب إلى Slack API
  2. اختر App ثم اذهب إلى Incoming Webhooks
  3. فعّل Webhooks وأنشئ واحد جديد
  4. اختر Channel (مثل #alerts)
  5. انسخ Webhook URL

🎮 Discord Webhook

📋 خطوات الحصول على Webhook:

  1. اذهب إلى Server Settings → Integrations
  2. اضغط على Webhooks → New Webhook
  3. اختر اسم للقناة (مثل "Alerts")
  4. انسخ Webhook URL

📧 تنبيه بالبريد

📋 الإعدادات:

  1. أدخل بريدك الإلكتروني
  2. سيتم إرسال إشعار عند توقف أي API
