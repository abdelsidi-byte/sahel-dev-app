# 🌐 Sahel Dev

> API Monitoring & Status Pages SaaS for Modern Teams

## 🎯 What is Sahel Dev?
Sahel Dev هو SaaS لمراقبة APIs وإنشاء صفحات الحالة (Status Pages).

## ✨ Features
- ⏱️ **Uptime Monitoring** - راقب APIs كل دقيقة
- 📊 **Status Pages** - ابن صفحات حالة احترافية
- 🔔 **Alerting** - تنبيهات Slack, Discord, Email, SMS
- 📈 **Analytics** - تقارير مفصلة

## 🛠️ Tech Stack
- **Frontend**: Next.js 14 + Tailwind CSS
- **Backend**: FastAPI (Python)
- **Database**: PostgreSQL (Neon)
- **Monitoring**: Custom poller + Health checks
- **Hosting**: Vercel (frontend) + Railway (backend)

## 🚀 Quick Start
```bash
# Clone
git clone https://github.com/saheldev/saheldev.git
cd saheldev

# Install
npm install

# Run
npm run dev
```
